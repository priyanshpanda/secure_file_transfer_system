Secure File Transfer with AES Encryption & Networking
``````````
to run this project:

step1: compile all
javac -cp "sqlite-jdbc-3.51.0.0.jar" *.java

Step2: run Server file
java -cp ".;sqlite-jdbc-3.51.0.0.jar" SecureServer

Step3:  run client file in another tab of cmd
java -cp ".;sqlite-jdbc-3.51.0.0.jar" SecureClient

ensure the jar file available in the file and check for version while compile /run

Step4: download DB Brower (SQLite) to view your database
